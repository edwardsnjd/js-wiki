# README

js-wiki is a browser based javascript application to display and navigate a wiki stored as markdown files.

## Features

- Hash fragment history to allow normal browser navigation
- HTML render of markdown files
- Linking between files (no hierarchy supported so relative links only work in flat structure?)
- Display of images
- Syntax highlighting for code blocks in common languages
- Visualise history?
- Search in files you've opened?
- Search in related files?
- Small footprint (inline scripts and styles into index.html)
